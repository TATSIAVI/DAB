Tools

Line Select the line tool from the top of the screen Left-click anywhere on the image to set a starting point, and left-click somewhere else to set an ending point Rectangle Select the rectangle tool from the top of the screen Left-click anywhere on the image to set a starting point, and left-click somewhere else to set an ending point Polygon Select the polygon tool from the top of the screen Left-click anywhere on the image to set a starting point Left-click on as many other places as you want to add points to the polygon Left-click on the starting point to finish drawing the polygon Polyline Select the polyline tool from the top of the screen Left-click anywhere on the image to set a starting point Left-click on as many other places as you want to add points to the polyline Right-click anywhere to create the last point and finish drawing the polyline

Manipulating Annotations

Moving Click and drag any point of the annotation to move it Deleting Select an annotation by clicking on any of its points OPTION 1) Press X/Delete/Backspace on the keyboard OPTION 2) Press the trash icon next to the annotation tag to the right of the window

Action Annotation

Adding annotations Click the NEW ACTION button Set the ID and the name of the action You should have unique IDs, but the action names may be the same. Marking actions to object Select the object that corresponds to a specific action Set its action ID to the corresponding action Action IDs should be unique to each object in the scene

Saving/Loading Annotations

Saving Click the bottom-right floating button with the plus icon to open a button list Click the button with the download icon Select your save path and click save Loading Click the hamburger menu located on the top-left of the screen Click Load Annotations Select your annotations JSON file and click open
